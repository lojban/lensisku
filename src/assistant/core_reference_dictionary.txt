## tutorial (notions; English ↔ English)

Using this dictionary ↔ Morphology (alphabet, brivla, cmavo, cmevla), bridi shape, and most particle usage are explained under **### selma'o** headings inside **## cmavo** (teaching notes as `#` lines before each class).
Alphabet ↔ Latin letters; ' = /h/ between vowels; . = pause before vowel-initial words; stress on the second-to-last vowel.
selbrivla (brivla) ↔ Relation/content word: consonant cluster within first five sounds, ends in a vowel (e.g. gleki, klama).
cmavo ↔ Particle: consonant + vowel (+ optional 'V sequences); may be run together (e.g. lenu, naku); grammar class = **selma'o** (subsection titles in **## cmavo**).
cmevla ↔ Name word: ends in a consonant; often written .name. with pauses matching dots.
Bridi ↔ A clause: sumti fill numbered places x₁ x₂ … of one selbri (relation).
